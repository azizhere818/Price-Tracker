# Price-Tracker
Developed a Python-based price monitoring system using Requests and BeautifulSoup that automatically tracks product prices, stores historical data in CSV files, and sends email notifications when price changes are detected.

## Features

- Scrapes product information
- Stores price history in CSV
- Sends automated email notifications
- Uses BeautifulSoup for HTML parsing
- Uses Yagmail for email delivery

## Technologies

- Python
- Requests
- BeautifulSoup
- Yagmail

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Configure email settings

3. Run

python price_tracker.py
